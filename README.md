# Website Traffic & User Interaction Analysis

## Data Analyst Internship Task – Alfido Tech

This project focuses on analyzing website traffic and user interaction data to understand engagement patterns, traffic trends, and content popularity using Python data analytics tools.

---

## Project Objective

The main objective of this project is to:

- Analyze website traffic and interaction logs
- Understand user engagement behavior
- Identify high-performing content and links
- Estimate sessions, users, and bounce rates
- Visualize traffic trends and interaction activity
- Recommend optimization strategies for Alfido Tech

---

Dataset Source:

---

## Key Findings

- Certain countries and cities generated significantly higher interaction activity.
- Popular artists and tracks received the highest engagement.
- Traffic activity varied across different dates.
- A few links accounted for the majority of interactions.
- Bounce behavior was observed in multiple sessions.

---

## Recommendations

1. Improve engagement on low-performing links.
2. Focus marketing campaigns on high-traffic regions.
3. Promote highly engaging artists and tracks.
4. Reduce bounce behavior through better navigation.
5. Analyze peak traffic periods for campaign optimization.

---

## Dataset Limitation

The dataset did not contain traditional web analytics fields such as session IDs, user IDs, landing pages, exit pages, or session duration. Therefore, sessions, users, and bounce rates were estimated using available interaction data.
